Welcome to Plex Info

A comprehensive tool for analyzing your Plex library with detailed media information, quality analysis, statistics, and health checks.

Features:

  • List library items with complete details
  • Analyze video quality and codec distribution
  • View library statistics and watch counts
  • Check library health and identify issues
  • View system and server information

List Library Items

View all items in your Plex library with complete details including file paths, quality, codecs, and subtitle information.

Health Check

Identify potential issues in your library.

Checks for:

  • Items with missing metadata
  • Low quality (SD) content
  • Items without subtitles
  • Very large files (>50GB)
  • Never watched content

Help & Documentation

Command Line Flags

Flag Description
--library Library name to analyze
--list-missing Show only items missing subtitles
--quality Analyze quality and codec distribution
--stats Show library statistics
--health Check library health
--system Display system information
--type Filter by media type (movie/episode)
--verbose Enable verbose logging

Running from Command Line

You can also run these commands directly from the terminal:

python plex_info.py --library "Movies" --quality
